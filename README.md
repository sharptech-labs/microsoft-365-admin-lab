<h1>Future Plans for This Microsoft 365 Lab</h1>

<p>
  This lab is designed to build real-world Microsoft 365 administration skills. Now that the tenant and test users are created, the next steps include:
</p>

<h3>1. User &amp; Identity Management</h3>
<ul>
  <li>Create, modify, and delete users</li>
  <li>Assign and remove licenses</li>
  <li>Reset passwords and manage MFA</li>
  <li>Test login flows with multiple test accounts</li>
</ul>

<h3>2. Group Management</h3>
<ul>
  <li>Create Microsoft 365 and security groups</li>
  <li>Manage group membership</li>
  <li>Assign groups to apps and resources</li>
</ul>

<h3>3. Exchange Online Administration</h3>
<ul>
  <li>Create shared mailboxes and distribution lists</li>
  <li>Configure mailbox permissions</li>
  <li>Test internal mail flow between users</li>
</ul>

<h3>4. Teams Administration</h3>
<ul>
  <li>Create teams and channels</li>
  <li>Manage team membership and permissions</li>
  <li>Test chat, meetings, and collaboration features</li>
</ul>

<h3>5. SharePoint &amp; OneDrive</h3>
<ul>
  <li>Create SharePoint sites</li>
  <li>Manage site permissions</li>
  <li>Test file sharing and access control</li>
</ul>

<h3>6. Security &amp; Compliance</h3>
<ul>
  <li>Enable MFA</li>
  <li>Review sign-in logs</li>
  <li>Explore basic alert policies</li>
</ul>

<h3>7. Device Management (Basic)</h3>
<ul>
  <li>Review device sign-ins</li>
  <li>Connect a Windows VM to Azure AD</li>
  <li>Explore compliance and device settings</li>
</ul>

<p>
  This roadmap will grow as I continue building hands-on experience with Microsoft 365 administration.
</p>
